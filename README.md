# FindSameRootWord
基于Python的自动查询同词根英语单词及释义，并将其写入Markdown笔记用以记录

代码中需要改动的部分是：
> WordRoot类中的self.path改为自己的路径下的.md文件

操作：
>改动上述部分后，运行py文件，输入想查询的单词即可
